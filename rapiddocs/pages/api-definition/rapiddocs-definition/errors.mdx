---
title: Errors in Rapiddocs Definition 
description: Add errors representing failed responses from API endpoints in Rapiddocs Definition.
---

Errors represent failed (non-200) responses from endpoints.

An error has:

- An HTTP status code
- A body type _(Optional)_

<CodeBlock title="user.yml">
```yaml
errors:
  UserNotFoundError:
    status-code: 404
    type: UserNotFoundErrorBody

types:
  UserNotFoundErrorBody:
    properties:
      requestedUserId: string
```
</CodeBlock>
